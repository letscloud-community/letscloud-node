# Letscloud Node

Api Wrapper to use [letscloud API](https://developers.letscloud.io/). This is built with typescript to better IntelliSense and learning.

## Getting Started

### Installing

```sh
# install with yarn
yarn add @letscloud/node

# install with npm
npm install @letscloud/node --save
```
